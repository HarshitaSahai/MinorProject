Please read the instructions carefully before each game in order to avoid any uncertainty.

1. The game consists of three pegs or towers with n disks placed one over the other.
2. You have to move the stack to another peg following these simple rules.
2(a). Only one disk can be moved at a time.
2(b). No disk can be placed on top of the smaller disk.
3. You have to move Disk from tower A(Original Disk) to C(Destination Disk) by using B(Auxilliary Disc).
4. The level of the game increase one by one and You have to move the original disk in the destination Disk with less time and less move for perfect score.
